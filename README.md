# Garage-Door
Raspberry Pi based garage door opener. This project consists of two parts, an API and a mobile web client.

Hardware:
	Raspberry Pi P1
	SunFounder 2 Channel DC 5V Relay Module
	Genie Intellicode Gitr-3 Remote
	Foscam FI8910W

Software:
	Raspbian GNU/Linux 8
	Apache/2.4.10
	Python 2.7.9
